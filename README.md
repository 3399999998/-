# **小熊优品**

小熊优品是基于阿里百川服务下的一款淘宝客APP,里面汇集了N家第三方平台的数据的商品,[轻淘客](http://www.qingtaoke.com/)、[好单库](https://www.haodanku.com/)、[淘客基地](http://www.tkjidi.com/)。（白话文就是一个领取优惠卷赚佣金的App）
程序还不是很完善,功能也很单一,也会有一些Bug，但是工作比较忙,后期还会继续更新,

主要分为五大板块：
+ 1 首页
+ 2 分类
+ 3 视频秀
+ 4 抢购
+ 5 我的（简单）

主要功能如下：
> 首页

> 轮播图、购物车、搜索、9.9包邮、大额卷、聚划算、活动、抽奖（当前版本未实现）

> 分类

>  各个商品模块分类显示,细化到具体物品

> 视频购

>  从第三方平台中获取有视频单的商品,并展示出来

> 抢购

>  抢购每天最新最好的商品,分为4个子模块,热卖、榜单、品牌、好实惠

> 我的

> 提供足迹模块入库,当前版本显示,点击用户头像可以进行登陆,注册等等
